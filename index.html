Create a Logging Class

The first thing I like to do when creating an Angular project is to create a common logging utility. You may call it whatever you like but for the sake of this tutorial FileAware.
All classes will extend this class. It can become tricky if you have other extended extended files. As an alternative you may just add this as a variable in the class using it and call it a log file.
